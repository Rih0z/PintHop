# PintHop 🍻

## Find your next pint. Find your next friends.

PintHop は、クラフトビール愛好家のためのソーシャルコミュニティアプリです。リアルタイムの共有、テーマ性のあるビアホッピング体験、そして写真ベースの超簡易記録システムを特徴とし、ビールを通じた人とのつながりを促進します。


## 🌟 主な特徴

- **ワンタップチェックイン**: 複雑な入力は不要。場所を選択するだけでチェックイン完了。
- **リアルタイム共有**: 誰がどこで何を飲んでいるかをリアルタイムで確認。
- **テーマ別ビアホッピング**: 「IPAの歴史を辿る旅」など、ストーリー性のあるブルワリー巡りルート。
- **写真ベースの記録**: タップリスト、ビール、ブルワリー外観の写真を簡単に共有。
- **コミュニティ活動**: サークル機能でイベント参加やメンバー間交流が簡単に。
- **資格認証システム**: BJCP、Ciceroneなどのビール資格の登録と認証。

## 🚀 他のアプリとの差別化ポイント

### 1. 「超簡易」設計哲学
- **Untappd vs PintHop**: Untappdが「何を飲んだか」の詳細記録を重視するのに対し、PintHopは「どこで誰と飲むか」という社会的体験を重視します。1人でもみんなでも楽しめるのがクラフトビール。入力は最小限で、写真撮影だけでも記録完了。
- **詳細は「任意」**: すべての詳細入力は完全に任意。初心者でも専門家でも自分のペースで利用可能。

### 2. リアルタイム社会的体験
- **既存のSNSと異なり**: 過去の記録ではなく「今」に焦点を当て、リアルタイムでの交流を促進。
- **「今から参加OK」機能**: 友達が今どこで飲んでいるかを確認し、即座に合流可能。

### 3. ストーリーテリングアプローチ
- **単なるリストではない**: ブルワリーを単にリスト化するのではなく、テーマやストーリーでつなげたビアホッピングルートを提供。
- **文脈のある体験**: 「シアトルのビール革命者たち」など、文脈のある巡り方を提案。

### 4. コミュニティファースト
- **地域コミュニティ統合**: シアトルビール部など、既存のビールコミュニティと連携。
- **オープンサークル設計**: 誰でも新しいビールコミュニティを作成・管理可能。

### 5. 専門知識の民主化
- **資格認証システム**: ビール専門家の知識を共有・活用できるプラットフォーム。
- **教育コンテンツ統合**: ビアスタイルガイド、テイスティング講座などの教育的要素。

### 6. ハイブリッドデジタル体験
- **オンラインからオフラインへ**: デジタル接点からリアルな出会いへスムーズに移行。
- **位置認識UI**: 現在地や時間に応じてコンテキストを理解し最適なUIを提供。

## 💻 技術スタック

### Webアプリ（フェーズ1）
- **フロントエンド**: React + TypeScript + Progressive Web App (PWA)
- **バックエンド**: Node.js + Express + TypeScript
- **データベース**: MongoDB Atlas
- **API設計**: GraphQL + Apollo Server
- **地図機能**: Google Maps API
- **リアルタイム通信**: Firebase Realtime Database

### モバイルアプリ（フェーズ3）
- **クロスプラットフォーム**: React Native + TypeScript
- **ネイティブ機能**: React Native Modules

## 🗺️ 開発ロードマップ
